「誠保一生」危疾保

守護您孩子的每一個人生階段，提供全面的危疾保障。點擊下方項目查看詳情。

👶 兒童疾病保障

+

保障額: 嚴重兒童病況: 基本保額的20% (23,400 港元)；次級嚴重兒童病況: 基本保額的5% (5,850 港元)

說明: 針對多種嚴重兒童病況和次級嚴重兒童病況，每項病況及每一組別各支付1次賠償。總賠償金額受限於基本保額的95%。

🩺 早期嚴重疾病保障

+

保障額: 冠狀動脈血管成形術/原位癌/早期甲狀腺或前列腺癌/次級侵害性惡性腫瘤: 基本保額的25% (29,250 港元)；其他45項早期嚴重病況: 基本保額的20% (23,400 港元)

說明: 本保障為每項早期嚴重病況支付1次賠償，惟冠狀動脈血管成形術及原位癌最多可分別索償2次。總賠償金額受限於基本保額的95%。

🏥 深切治療保障

+

保障額: 合資格的深切治療部留醫: 基本保額的20% (23,400 港元)。如於香港或澳門境外，調整至基本保額的10%。

說明: 針對因醫療需要於深切治療部連續留醫3天或以上的情況。本保障只支付1次賠償。總賠償金額受限於基本保額的95%。

🛡️ 嚴重疾病保障

+

保障額: 56項嚴重病況: 基本保額的100% (117,000 港元) + 特別紅利的面值 (如適用)

說明: 涵蓋56項嚴重病況，只支付1次賠償。支付後，兒童疾病保障、早期嚴重疾病保障、深切治療保障、身故保障及退保價值將終止。

📈 升級保障

+

保障額: 於第10個保單周年日或之前確診56項嚴重病況或身故: 基本保額的50% (58,500 港元)

說明: 本保障只支付1次賠償。於升級保障完結前或後之1個月內，您可以將升級保障轉換至一份我們指定並備有現金價值的全新壽險計劃，而毋須提供額外健康證明。

➕ 嚴重疾病延伸保障

+

保障額: 於86歲(翌年歲)前確診癌症、心臟病發作、中風、阿耳滋海默氏症/不可還原之器質性腦退化疾病、柏金遜病及其他48項嚴重病況: 基本保額的100% (117,000 港元)。最高保障額可達基本保額的900%。

說明: 嚴重疾病保障已支付後，仍可就特定嚴重病況提出索償，惟需符合時間間隔要求。

🧠 嚴重腦退化症或柏金遜病終身年金

+

保障額: 每年基本保額的6% (7,020 港元)

說明: 於嚴重疾病保障或嚴重疾病延伸保障已支付後，若確診嚴重腦退化症或柏金遜病，每年支付直至受保人身故為止。

🕊️ 人壽延伸保障

+

保障額: 與首次確診嚴重病況相隔最少一年後並於86歲(翌年歲)前身故: 基本保額的100% (117,000 港元)

說明: 提供身故保障，即使在嚴重疾病保障支付後，若符合條件仍可獲得。

💰 保費豁免保障

+

保障內容:

  • **早期危疾保費豁免保障:** 當早期嚴重疾病保障或嚴重兒童病況的兒童疾病保障賠償須支付及/或已付訖時，基本計劃往後12個月之到期保費總額將被豁免。
  • **危疾保費豁免保障:** 當嚴重疾病保障賠償須支付及/或已付訖時，基本計劃日後的所有保費將被豁免。
  • **親子保費豁免保障:** 當保單持有人或其於保單文件紀錄下之配偶身故時，由下一個保費到期日起直至受保人26歲(翌年歲)的所有基本計劃保費將被豁免。
